# Credit Card Intelligence Engine

A terminal-first, data-driven decision engine for Indian credit cards,
built to help users:

- Decide which credit card to apply for
- Decide which credit card to use for a transaction
- Optimize monthly spending across cards
- Redeem reward points for maximum real-world value

## Design Principles
- Deterministic logic > AI hype
- Transparency over black-box decisions
- Fewer cards, higher accuracy
- Easy extensibility (add more cards later)

- Version control: GitHub (single source of truth)
